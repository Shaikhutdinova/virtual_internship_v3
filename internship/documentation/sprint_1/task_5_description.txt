Папка tmp:
Папка tmp - это временный каталог, используемый Gradle для хранения промежуточных файлов во время сборки.

Что содержится в папке tmp:
• Загруженные зависимости: Gradle загружает зависимости проекта в папку tmp, прежде чем скопировать их в окончательный
  каталог с зависимостями.
• Скомпилированные файлы: Промежуточные скомпилированные файлы также могут храниться в папке tmp перед тем, как их
  скопировать в каталог build/classes.
• Файлы для тестов: Временные файлы, созданные для выполнения тестов, могут храниться в папке tmp.
• Другие временные файлы: Gradle может создавать другие временные файлы, необходимые для выполнения различных задач
  сборки.

Важно отметить:
• Содержимое папки tmp может меняться в зависимости от конфигурации вашего проекта и используемых плагинов Gradle.
• Папка tmp обычно удаляется после успешной сборки проекта.
• В некоторых случаях папка tmp может быть очищена во время сборки, чтобы предотвратить накопление временных файлов.
