Преимущества Maven структуры:

• Упрощает процесс сборки: Maven автоматически управляет зависимостями, сборкой, тестированием и упаковкой проекта.
• Улучшает читаемость: Структура проекта становится более понятной и логичной.
• Снижает риск ошибок: Стандартная структура помогает избежать ошибок, связанных с неправильным размещением файлов.
• Повышает производительность: Автоматизация сборки и управления зависимостями освобождает время разработчиков.

Maven структура проекта состоит из:
- src - директория с кодом
- src/main - в директории лежит продакшн код
- src/main/java - Java продакшн код (классы)
- src/main/resources - продакшн файлы ресурсов
- src/test - в директории лежит тестовый код
- src/test/java - тесты написанные на Java
- src/test/resources - тестовые файлы ресурсов
